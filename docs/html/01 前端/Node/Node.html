包管理器 npm

sh
# 生产环境
npm install express --save
sh
# 开发环境
npm install express --save-dev
npm install express -D
sh
# 全局安装
npm install express -g
sh
# 临时安装，不添加到package.json
npm install express

如果 package.json 中的依赖项指定了 ^1.2.3，npm update 会更新到 1.x.x 的最新版本，但不会更新到 2.0.0。

sh
npm update express
sh
# 卸载
npm uninstall express

# 版本
npm install express@1.0.0

# 安装所有依赖
npm install

npm ci 会完全依照 package-lock.json 文件来安装依赖包，确保安装的版本与锁定文件中完全一致。

sh
# 生产环境安装
npm install --production
npm ci
sh
# scripts 指令
npm run start

npx用于临时执行包的命令或本地命令而不需要安装此包：

sh
npx eslint . --ext .ts
npx create-react-app my-app
