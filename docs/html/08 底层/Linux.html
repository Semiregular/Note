概述

用户

shell
# 添加用户
sudo useradd username

# 修改密码
sudo passwd username

# 添加组
groupadd groupname

# 添加用户到组
sudo usermod -aG groupname username

# 查看用户组所有成员
getent group groupname

# 查看所有用户组
getent group

# 删除用户及其用户目录
userdel -r username

# 查看所有用户
getent passwd

# 切换用户
su - username
su username

文件权限

第一个权限字母表示文件类型，第二个权限字母表示文件所有者的权限，第三个权限字母表示文件所在组的权限，第四个权限字母表示其他用户的权限。

对于目录而言，w 表示可以删除目录下的文件，x 表示可以进入该目录。

=== "文件类型"

第一个字母|文件类型

-|-

d|目录

l|符号链接

b|块设备文件

c|字符设备文件

s|套接字文件

p|管道文件

-|文件

=== "权限权值"

权限|权值

-|-

r|4

w|2

x|1

=== "常见权限"

数字|权限

-|-

7|rwx

6|rw-

5|r-x

4|r--

3|-wx

2|-w-

1|--x

0|

shell
# 修改文件用户组
chgrp groupname filename

# 修改目录下所有文件用户组
chgrp -R groupname filename

# 修改文件所有者
chown username filename

# 修改目录下所有文件所有者
chown -R username filename

# 修改文件权限
chmod 777 filename
chmod a+x filename

# 修改目录下所有文件权限
chmod -R 777 filename

!!! note "注意"

若目录用户具有权限 rwx , 目录下某个文件用户没有任何权限，则用户无法访问该文件，但可以删除该文件。

命令

ls

shell
# 详细信息显示
ls -l

# 时间排序
ls -t

# 显示隐藏文件
ls -a

rm

shell
# 删除文件,会询问
rm -i filename

# 递归删除文件
rm -r filename

# 强制删除目录，无警告
rm -f directory

cp

shell
cp original_file new_file

mv

可以用来重命名

shell
mv original_file new_file

cat

shell
# 显示行号
cat -n file

# 列出特殊字符
cat -A file

head

shell
# 显示前10行
head -n 10 file

# 除了最后10行
head -n -10 file

tail

shell
# 显示最后10行
tail -n 10 file

# 除了前10行
tail -n +10 file

more

空格翻页，enter下一行，q退出，b上一页

shell
# 分页显示
more file

file

file 查看文件类型

shell
file main.cpp

cut

cut 剪切字符串。

参数 -d 后的字符表示字段分隔符，默认为空格，表示在处理文件的内容时，将以其作为字段分隔符。

参数 -f 后的字符表示要提取的字段，默认为1，表示从第一列开始提取。

shell
cut -d ' ' -f2 file

wc

wc ，用于计算文件中的文本行数、单词数和字符数。

shell
# 行数、单词数、字符数
wc -l file
wc -w file
wc -c file
