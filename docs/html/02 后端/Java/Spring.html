方法

@Controller

@Contoller 使得Spring容器能够识别并注册这个类作为处理HTTP请求的组件

  • @Controller

@Controller注解用于标注控制层组件，用于处理请求并返回响应数据，一般用于返回视图

java
@Controller
public class SampleController {

    @GetMapping("/hello")
    public String handleHelloRequest(Model model) {
        model.addAttribute("greeting", "Hello, World!");
        // 返回视图名，对应一个HTML模板或JSP页面
        return "hello";  
    }
}
   
@Controller
public class UserResource {
    private final UserService userService;

    @Autowired
    public UserResource(UserService userService) {
        this.userService = userService;
    }

    @ResponseBody
    @GetMapping("/users/{id}")
    public User getUser(@PathVariable Long id) {
        // 直接返回User对象，将被自动序列化为JSON等格式
        return userService.findById(id);  
    }
}
  • @RestController

@RestContoller 等于@Controller和@ResponseBody的组合，表明返回的是JSON数据

java
@RestController
public class UserResource {
    private final UserService userService;

    @Autowired
    public UserResource(UserService userService) {
        this.userService = userService;
    }

    @GetMapping("/users/{id}")
    public User getUser(@PathVariable Long id) {
        // 直接返回User对象，将被自动序列化为JSON等格式
        return userService.findById(id);  
    }
}

@RequestMapping

  • @RequestMapping

@RequestMapping注解用于映射请求到控制器方法，可以同时映射多个请求，在一个方法上可以使用多个 @RequestMapping 注解，它们的效果相当于将所有注解的属性合并在一起。

java
@RequestMapping(path = "/{id}", method = RequestMethod.GET)
public String getUserDetails(@PathVariable Long id, Model model)

@RequestMapping(method = RequestMethod.GET)
public String listUsers(Model model)

/**
 * Consumes 和 Produces：指定请求和响应的内容类型（MIME类型）。
 * 例如，可以指定只处理application/json类型的POST请求，
 * 并返回application/json类型的响应。
 */
@RequestMapping(path = "/users", method = RequestMethod.POST,
                consumes = MediaType.APPLICATION_JSON_VALUE,
                produces = MediaType.APPLICATION_JSON_VALUE)
public User createUser(@RequestBody User user)

@RequestParam

@RequestParam注解用于获取请求参数，可以指定参数名称、默认值、是否必填等。

java
// 指定要绑定的请求参数名。当方法参数名与请求参数名相同时，可以省略value属性。
@GetMapping("/search")
public List<Product> searchProducts(
    @RequestParam("keyword") String keyword)

@GetMapping("/search")
public List<Product> searchProducts(
    @RequestParam(value = "page", defaultValue = "1") int pageNumber) 

// 指定该请求参数是否必须存在。默认值为true，即请求中必须包含该参数
@GetMapping("/search")
public List<Product> searchProducts(
    @RequestParam(value = "sortOrder", required = true) String sortOrder)

@PathVariable

@PathVariable注解用于获取请求路径中的变量，可以指定变量名称。

java
// 指定要绑定的路径变量名。当方法参数名与路径变量名相同时，可以省略value属性
@GetMapping("/users/{userId}")
public User getUserDetails(@PathVariable Long userId) 
 
@GetMapping("/books/{bookId}/chapters/{chapterNumber}")
public Chapter getChapterDetails(
        @PathVariable("bookId") Long bookId, 
        @PathVariable("chapterNumber") int chapterNumber) 

@GetMapping("/books/{bookId}/chapters/{chapterNumber}")
public Chapter getChapterDetails(
        @PathVariable(value = "bookId", required = true) Long bookId,
        @PathVariable(value = "chapterNumber", required = false, defaultValue = "1") int chapterNumber)
