前言

计算机行业或许与其他行业都不同，在于计算机技术并不是一种传统的技术：

  • 从源头来看，计算机行业不是凭空出现，而是在众多行业的基础上建立并发展起来；同样，计算机的发展也推动了众多领域。可以说，计算机技术更像是电力的具体应用，将生产力和我们的生活紧密联系在一起，而且这种联系易于观察且普遍。
  • 从传统来看，计算机行业推崇开源精神，主张大家交流协作；同样，万维网的初衷也正是让资源能够共享。这种理念我们姑且认为是“草根精神”，鼓励大家发声，共同进步。
  • 从发展来看，正是由于上述两点，计算机技术的发展日新月益，迭代速度快，往往一些有名的技术框架过一段时间就会消弭，被新的技术取代；同时，市面上的产品也多种多样。

作为计算机行业的从业者，我觉得意识到以下几点很重要：

  • 保持头脑极度开放，快速拥抱新技术。这要求我们深刻理解使用的技术，而不是浮于表面；不要惧怕新的技术和观念，而要提醒我们自己，所有流程都不是理所应当，所有传统技术都有待商榷。
  • 懂得架构。人人都应首先是一名架构师，否则，我们终会陷于某些无意义的工作。试着闭上双眼，像一个比特一样过一下流程，我们会清楚自己的位置。
  • 从产品角度思考。做软件必须考虑用户如何更加便利的使用，而不是只从开发的角度思考；同时，我们要观察市面上的产品，不必懂得如何赚钱，但要明白趋势。
