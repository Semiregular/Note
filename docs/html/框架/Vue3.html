以下是Vue3的主要更新点：

性能提升

打包大小显著减少（41%），这有助于提高应用加载速度。

新架构与API变化
  • Composition API：

引入了全新的Composition API

  • 响应式系统改进：

使用ES6的Proxy代替Object.defineProperty实现更完整的响应式代理，增强了对对象属性添加/删除等操作的监听能力。

  • Tree-Shaking支持：

通过更好的模块化设计，Vue3能够更好地支持treeshaking，从而减小生产环境下的打包体积。

  • 组件生命周期调整:

Vue3移除了部分生命周期钩子如beforeCreate和created

  • 其他新特性与优化:

ref和reactive等核心API用于管理组件状态。

支持TypeScript编写，官方提供了丰富的类型定义文件。

  • 工具链支持:

Vue CLI 和 Vite 的更新，Vite 提供了更快的开发体验，利用原生ES模块实现快速热更新。

修改 main.js 如下：

javascript
import { createApp } from 'vue'
import App from './App.vue'
import router from './router/index' //导入路由表

const app = createApp(App);
app.use(router).mount('#app'); //使用路由导航
